Test Steps - Configuration Group - Search
=====================================
Module  : Web Dashboard
Menu    : Configuration Management > Configuration Group
Feature : Search Configuration Group

Steps:
1. Login to the application
2. Click "Configuration Management" menu
3. Click "Configuration Group" sub-menu
4. Read search_configuration_group.xlsx, fill the "Configuration Group Name" search field according to each test case row, click the "Search" button, and take a screenshot
   - Repeat this process until all rows have been executed
5. For the negative case, verify the Expected_Result text is visible on the page (e.g. "No data registered yet")
6. For the positive case, click the "Details" link in the search result and validate the following fields on the detail page match the values in the Excel: Configuration Group Name, Description, Status, Ecosystem Coverage
7. Screenshot evidence is saved per test case using the TC_ID as the filename (e.g. TC-001.png)
8. Only execute the test cases defined in search_configuration_group.xlsx — do not create new ones
9. Generate the test report
