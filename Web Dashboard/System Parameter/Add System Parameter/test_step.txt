Test Steps - System Parameter - Add New
=====================================
Module  : Web Dashboard
Menu    : System Setting Management > System Parameter
Feature : Add System Parameter

Steps:
1. Login to the application
2. Click "System Setting Management" menu
3. Click "System Parameter" sub-menu
4. Click "Add System Parameter" button
5. Read system_parameter.xlsx, fill all of the fields according to each test case row, click the Save button, and take a screenshot
   - Repeat this process until all rows have been executed
6. For the positive case, add the datetime to Parameter Code to make it unique
7. Screenshot evidence is saved per test case using the TC_ID as the filename (e.g. TC-001.png)
8. Only execute the test cases defined in system_parameter.xlsx — do not create new ones
9. Generate the test report
